Anything that appears within "(b)"..."(/b)" element, is displayed in bold as shown below:

Output :

  • The HTML Bold "b" tag is used to display the written text in bold format.

  • It is strictly a presentational element. If you want to show your text in bold letters and not have real semantic meaning, then put it within "b"......."/b" tag.

  • To define text with strong importance, place the text inside "bold" and "strong" tags.
